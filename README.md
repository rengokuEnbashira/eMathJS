# eMathJS
This is a simple mathematical javascript library. 
## ematrix (Matrix class)
### Fields
- rows, cols
- data
### Methods
- zeros
- ones
- random
- eye
- to_string
- add
- diff
- mul
- div
- dot
- scale
- trace
- diag
- transpose
- lu
- qr
- det
- inv
- eig
## epoly (Polynomial class)
### Fields
- degree
- coeffs
### Methods
- to_string
- eval
- derivate
- integrate
- add
- diff
- mul
- div
- scale
- roots

### Methods
- linspace
- euler
- rk4
- solve
## enonlinear (Non-linear equation solver class)
### Methods
- newton
- secant
- bisection
